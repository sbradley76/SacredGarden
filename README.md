# The Sacred Garden — V2

A refined Vite + React storefront concept for a premium smoke shop brand.

## What changed in this pass
- upgraded placeholder graphics with moodier, more realistic-looking product/category art
- stronger typography proportions and cleaner spacing
- reduced pill rounding in navigation, chips, and buttons
- interactive mock cart drawer
- quick-view product modal
- working product add buttons and category filters
- cleaner shop browsing flow without Shopify wiring yet

## Run locally

```bash
npm install
npm run dev
```

## Notes
- `node_modules` and lockfiles are intentionally omitted from this zip.
- Images in `src/assets` are still placeholder art direction assets, but they are built to feel more realistic and premium than V1.
- Shopify wiring can be layered onto the existing product and cart patterns in the next pass.
